# Spring Cloud Examples

This project demonstrates various implementations of Spring Cloud features using different service discovery mechanisms.
It serves as a practical guide for implementing microservices architecture using Spring Cloud.

## Modules

- **spring-boot-cloud-nodiscovery**: Example implementation without service discovery
- **spring-boot-cloud-eureka**: Implementation using Netflix Eureka as service discovery
- **spring-boot-cloud-kubernetes**: Implementation using Kubernetes native service discovery

## Requirements

- Java 17 or higher
- Maven 3.6+
- Docker (for Kubernetes examples)

## Versions

- Spring Boot: 3.5.4
- Spring Cloud: 2025.0.0
- Spring Boot Admin: 3.2.3

## Project Structure

This is a multi-module Maven project that showcases different approaches to implementing microservices using Spring
Cloud.

For detailed implementation of each module, please refer to their respective README files.
